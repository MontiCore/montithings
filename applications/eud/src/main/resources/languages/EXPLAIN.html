This project is a simple example for the end user development feature for MontiThings. It consists of 4 Components:

  • "Machine", which is the parent component.
  • "numGen", which outputs a number that constantly counts up. This component also receives the final result and logs it.
  • "calc", which is a component that can receive an end-user model and does some math on the incoming variable, based on this uploaded model.
  • "print", which is a component that can receive an end-user model and creates different strings dependent on the in the model specified parameters.
