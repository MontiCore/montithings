<h1> CalculatorLang</h1>
<pre>
This language can represent simple calculations on an incoming variable.
For Example:
    + 100;
    * 10;

If the incoming variable is 5 this would calculate (5 + 100) * 10 and return the result.

Rules:
</pre>
<ul>
  <li>Every line has to end with a ";"</li>
  <li>Only allowed charcters are "*" and "+"</li>
  <li>Only numbers in &#8484 are allowed</li>
</ul>
